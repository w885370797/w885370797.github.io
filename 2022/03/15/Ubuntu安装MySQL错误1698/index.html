sudo apt-get install mysql-server

netstat -tap | grep mysql
在愉快的执行完这两条命令之后，duang的一下给我来个报错，非常难受。经过仔细查阅相关资料找到了解决办法。

解决方法：

在ubuntu的terminal（也即终端）上输入sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf然后在该配置文件中增加skip-grant-tables

之后重启mysql服务 service mysql restart

很明显现在我们输入mysql -u root -p已经可以进入了，
